## 🎓 YouTubeスタイル学習

好きなYouTubeチャンネルの編集スタイルを学習できます:

```javascript
// learn-style.js を作成
import AutoEditPipeline from './src/core/index.js';

const pipeline = new AutoEditPipeline();

// YouTubeチャンネルの動画URLを指定
await pipeline.learnStyle([
  'https://www.youtube.com/watch?v=VIDEO_ID_1',
  'https://www.youtube.com/watch?v=VIDEO_ID_2',
  'https://www.youtube.com/watch?v=VIDEO_ID_3',
], 'my-channel-style');

console.log('スタイル学習完了！');
```

実行:
```bash
node learn-style.js
```

学習したスタイルで編集:
```bash
node src/core/index.js your-video.mp4 --style-name my-channel-style
```

## ⚙️ 設定のカスタマイズ

`config/default.config.js` で細かい調整が可能:

```javascript
autoCut: {
  silenceThreshold: -40,       // 無音検出の閾値（小さいほど敏感）
  minClipDuration: 1.0,        // 最小クリップ長（秒）
  fillerWords: ['えー', 'あー'], // 除去するフィラーワード
}

caption: {
  maxCharsPerLine: 20,         // 1行の最大文字数
  defaultStyle: {
    fontSize: 48,
    color: '#FFFFFF',
    position: 'bottom',        // 'top', 'middle', 'bottom'
  },
}
```

## ❓ トラブルシューティング

### エラー: "OPENAI_API_KEY is not set"
→ `.env`ファイルが正しく設定されているか確認

### エラー: "ffmpeg not found"
→ FFmpegをインストール: `brew install ffmpeg` (macOS)

### カットが多すぎる/少なすぎる
→ `config/default.config.js`の`silenceThreshold`を調整
